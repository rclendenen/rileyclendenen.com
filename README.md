## üöÄ Getting Started

### Prerequisites
- A modern web browser
- A local web server (optional, for development)

### Installation

1. **Clone or Download** the project files
2. **Open** `index.html` in your web browser
3. **Customize** the content with your personal information

### Customization Guide

#### Personal Information
- Replace placeholder text in `index.html` with your actual content
- Update the hero section with your name and professional title
- Add your real photo URLs or replace with local images
- Update contact information and social media links

#### Portfolio Projects
- Replace the three portfolio examples with your actual projects
- Update project images, descriptions, and technology tags
- Add real live links to your projects
- Include screenshots or mockups of your work

#### Testimonials
- Replace placeholder testimonials with real client feedback
- Add client names and titles (with permission)
- Include headshots if available

#### Styling
- Colors can be adjusted in the CSS variables at the top of `styles.css`
- Fonts can be changed by updating the Google Fonts import
- Layout spacing can be modified using the CSS custom properties

### Color Customization
The color scheme is defined using CSS custom properties in `:root`. To change colors:

```css
:root {
    --sage-green: #9CAF88;        /* Primary accent */
    --sage-green-light: #B8C7A6;  /* Light variant */
    --sage-green-dark: #7A8B6B;   /* Dark variant */
    --sage-green-muted: #E8F0E3;  /* Background tint */
    /* ... other colors */
}
```
